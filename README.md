## Getting Started

### Prerequisites

-   nodejs

### Installation

1. Clone the repo
    ```sh
    git clone https://github.com/TheIthorian/Meal-Planner.git
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Run the setup script
    ```sh
    node setup.js
    ```
